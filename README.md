# 🚀 AWS 3-Tier Architecture Project

---

## 📌 Description
This project demonstrates a 3-tier architecture deployed on AWS. The architecture separates the application into presentation, application, and database layers to improve scalability, security, and maintainability.

---

## 🏗️ Architecture Overview
- Presentation Layer (Frontend)
- Application Layer (Backend)
- Database Layer (RDS)

---


## 🚀 **In this Project, I have used the following AWS services:**

- **VPC**  
- **S3**  
- **IAM**  
- **EC2**  
- **RDS**  
- **Route 53**
